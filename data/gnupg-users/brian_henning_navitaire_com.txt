I believe that the pgp_outlook_compat option is only available if you patch
mutt (which I think debian does in it's standard mutt package).
For those that need to generate inline pgp messages with a text/plain
content-type, I'd recommend Dale Woolridge's patch[1] with 1.4 or use the
latest 1.5 devel versions, which send text/plain for pgp traditional
messages by default.
