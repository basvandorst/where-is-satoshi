@_subject: Implications of using insecure memory 
The security risks from using Windows are such that I would be very leery of relying on an encryption application to provide any security on a Windows computer.
As far as GNU/Linux, it's simple to get rid of this warning, you can simply run "chmod u+s `which gpg`" (GnuPG will detect that it's running setuid root and drop privileges).
